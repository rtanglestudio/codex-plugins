# Privacy Policy

Product UI Architect is distributed as a local Codex plugin repository.

## Data handling

- This repository does not operate a hosted service.
- This repository does not collect analytics, telemetry, or personal data by itself.
- Any data processing happens inside the user's local Codex environment and follows the behavior of Codex, GitHub, and any other tools the user chooses to connect.

## Third-party services

If you use this plugin with Codex, GitHub, or other connected tools, those services may process data according to their own policies.
