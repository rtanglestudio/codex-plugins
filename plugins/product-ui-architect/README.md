# Product UI Architect

Product UI Architect is a Codex plugin that improves the design quality of generated SaaS interfaces, dashboards, AI workflow tools, and marketing pages.

It is built for one job: make Codex stop producing generic floating-card startup UI and start producing calmer, more structured, more usable product surfaces.

## Core outcomes

- stronger information architecture
- more task-first layouts
- better defaults for app shells and dashboards
- less card soup, glow, and ornamental chrome
- clearer hierarchy, density, and accessibility

## Skill trigger

Use `$product-ui-architect`.
